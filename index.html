🚀 Configuration PrayLink

Serveur Google Apps Script + Google Sheets

📊 Étape 1: Créer Google Sheets

1. Allez sur Google Sheets

2. Créez un nouveau document nommé "PrayLink Database"

3. Créez ces feuilles (onglets) :

📋 Feuille "Users"

A1: id
B1: username
C1: platform
D1: profileEmoji
E1: createdAt

💬 Feuille "Messages"

A1: id
B1: recipient
C1: type
D1: content
E1: sender
F1: createdAt

📖 Feuille "Verses"

A1: id
B1: text
C1: reference
D1: category
E1: language

📊 Feuille "Stats"

A1: metric
B1: value
C1: updatedAt

⚙️ Étape 2: Google Apps Script

1. Allez sur Google Apps Script

2. Créez un nouveau projet nommé "PrayLink API"

3. Remplacez le code par défaut par celui ci-dessous :

4. Remplacez YOUR_SHEET_ID par l'ID de votre Google Sheets

5. Cliquez sur "Déployer" → "Nouveau déploiement"

6. Type: "Application web"

7. Accès: "Tout le monde"

8. Copiez l'URL du déploiement
